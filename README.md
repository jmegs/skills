# skills

this is a collection of skills. there are many like it but this one is mine.

## contents

- **bounded-context-research**: Research open-ended questions with cited sources using web search.
- **grill-me**: Interview you relentlessly about a plan or design, walking each branch of the decision tree until shared understanding is reached.
- **rwsdk**: Build full-stack apps with RedwoodSDK on Cloudflare Workers using TypeScript, React Server Components, D1, R2, and server functions.
- **youtube-summarizer**: Summarize YouTube videos by extracting captions and producing a terse, chronologically ordered report of key points.

## how to install

```sh
# or bunx, pnpm dlx, whatever you use
npx skills add jmegs/skills
```
